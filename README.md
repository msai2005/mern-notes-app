# Notes App (MERN Stack)

A simple CRUD Notes App built with the *MERN stack* (MongoDB, Express.js, React, Node.js).

## Features
- Add new notes (title + content)
- View all notes
- Edit notes
- Delete notes

## Project Flow
1. The frontend (React) allows users to add, view, edit, and delete notes.
2. The backend (Express + Node) provides REST APIs.
3. Data is stored in MongoDB.
4. Axios connects frontend and backend.

## How to Run (Conceptual)
- Start backend: npm run server
- Start frontend: npm start
